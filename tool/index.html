# 大前端工具集合

  • 常用web前端工具集 - 很多，有你想要的
  • 也许是史上最全的前端资源大汇总 - 尽情的淘吧
  • 大前端工具集，只记录有用的
  • 前端常用插件、工具类库汇总，不要重复造轮子啦！！！ - 前端常用插件、工具类库汇总，不要重复造轮子
  • Gulp - 用自动化构建工具增强你的工作流程！
  • Webpack - 是一个前端资源加载/打包工具
  • Babel - Babel 是一个 JavaScript 编译器,今天就开始使用下一代的 JavaScript 语法编程吧！
  • ESlint - ESLint 是在 ECMAScript/JavaScript 代码中识别和报告模式匹配的工具，它的目标是保证代码的一致性和避免错误。
  • Jest,好用的测试框架，内置断言库。 另一款测试框架mocha也不错,但要搭配断言库使用，如chaijs
  • momentjs - JavaScript 日期处理类库
  • backbonejs - 是一种帮助开发重量级的javascript应用的框架。主要提供了3个东西：1、models(模型) 2、collections(集合) 3、views(视图)，当然也有路由,它允许你以MVC（模块，视图和控制器）的结构方式去架构你的JS代码.
  • underscorejs - Underscore是一个JavaScript库，提供了许多有用的工具方法，集合操作，而无需扩展任何内置对象
  • Loadsh - 是一个一致性、模块化、高性能的 JavaScript 实用工具库。
  • jQuery - jQuery 是一个高效、精简并且功能丰富的 JavaScript 工具库。它提供的 API 易于使用且兼容众多浏览器，这让诸如 HTML 文档遍历和操作、事件处理、动画和 Ajax 操作更加简单
  • swiper - 开源、免费、强大的触摸滑动插件
  • fullPage - 好用的全屏股东插件
  • photoswipe - 图片画廊插件
  • cropper - 国人开发的图片裁剪库
  • echarts - 数据可视化-图表插件
  • ighcharts - 数据可视化-图表插件
  • Chart.js - 适用于设计人员和开发人员的简单而灵活的JavaScript图表
  • axios - 众所周知的易用、简洁且高效的http库，基于pormise的API，node和浏览器全支持，平台切换无压力,支持拦截器等高级配置
  • zTree - zTree 是一个依靠 jQuery 实现的多功能 “树插件”。优异的性能、灵活的配置、多种功能的组合是 zTree 最大优点

Swiper/PhotoSwipe/fullPage 有这仨库，微信里常见的H5页完全不是问题
