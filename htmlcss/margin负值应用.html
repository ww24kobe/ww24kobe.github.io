# margin负值应用

可以使用margin负值解决相邻元素边框重合问题，如果给子盒子设置左右1px像素的边框，那么相邻的两个盒子由于浮动（左浮）排在一行就会显示2px像素的边框,利用margin-right负值会把盒子移动到左边从而相邻元素的边框就会合在一起显示我们期望的1px。

代码如下

<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>Document</title>
	<style>
		
		.parent {
			width: 300px;
			height: 300px;
			border: 2px solid red;
			margin: 20px auto;
			
		}

		.son {
			width: 99px;
			height: 100px;
			border: 1px solid #666;
			margin-right: -1px;
			float: left;
		}

		.son:hover {
			border: 1px solid orange;
			position: relative;  /*提升高度，压在标准流上面*/
			
		}
	</style>
</head>
<body>

	<div class="parent">
		<div class="son"></div>
		<div class="son"></div>
		<div class="son"></div>
	</div>
	
</body>
</html>
